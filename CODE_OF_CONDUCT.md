# Code of Conduct

## Our standard

We want the Mosaicora plugin community to be welcoming, respectful, practical,
and safe for contributors of all experience levels.

Examples of behavior that support a positive environment include:

- showing empathy and patience
- giving and receiving constructive feedback
- assuming good intent while still being clear about technical concerns
- focusing on what helps the project and the people using it
- respecting different viewpoints, backgrounds, and levels of experience

Examples of unacceptable behavior include:

- harassment, discrimination, or intimidation
- abusive, insulting, or demeaning language
- publishing private information without explicit permission
- trolling, bad-faith arguments, or deliberately disruptive behavior
- sexualized language or unwelcome sexual attention

## Scope

This code of conduct applies in:

- GitHub issues and pull requests
- project discussions and documentation
- any other project-managed community spaces

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of
conduct. They may remove, edit, or reject comments, commits, code, wiki edits,
issues, and other contributions that violate these standards.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers
privately through GitHub instead of opening a public issue.
